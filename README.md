# Financial Transactions Dashboard 📊

## 📌 Overview
This project presents a comprehensive *Financial Transactions Dashboard* built using Microsoft Excel.  
It transforms a dataset of 5,000 transactions into clear, interactive insights for better understanding of revenue and performance.

---

## 🔍 Key Metrics
- Total Revenue: $24.7M  
- Total Transactions: 5,000  
- Total Cashback: $253.3K  
- Total Fees: $125.9K  

---

## 💡 Key Insights
- Urban areas are the main revenue drivers, contributing over $17M.  
- Android users represent the highest number of transactions (2,995).  
- Revenue peaked in July, followed by a slight decline until September.  
- Bank Transfers are the most trusted and frequently used payment method.  

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Interactive Slicers  
- Data Cleaning & Preparation
